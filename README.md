# Professional Freelancer Portfolio

A modern, responsive portfolio website for a freelancer specializing in web development, graphic design, and video editing.

## Features

- **Modern Design**: Clean, professional layout following the 60-30-10 color rule
- **Dark/Light Mode**: Toggle between light and dark themes
- **Responsive Layout**: Fully responsive design for all device sizes
- **Smooth Animations**: Scroll-triggered animations and smooth transitions
- **Project Filtering**: Filter projects by category (web, design, video)
- **Interactive Elements**: Hover effects, form validation, and smooth scrolling

## Color Palette

The website follows the 60-30-10 design rule:

### Light Mode:
- 60%: White (#ffffff) - Background
- 30%: Light Gray (#f8f9fa) - Secondary elements
- 10%: Blue (#4a6cf7) - Accents, buttons, highlights

### Dark Mode:
- 60%: Dark Gray (#121212) - Background
- 30%: Charcoal (#1e1e1e) - Secondary elements
- 10%: Light Blue (#6a8cff) - Accents, buttons, highlights
